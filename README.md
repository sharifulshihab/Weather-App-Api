# 🌦️ Weather App (PyQt5 Desktop Application)

A beautiful desktop weather application built using Python and PyQt5.  
It provides real-time weather updates, animations, and a 24-hour forecast chart using OpenWeather API.


## 🚀 Features

- 📍 Auto location detection
- 🌡️ Real-time weather data
- 🔄 Celsius / Fahrenheit toggle
- 📊 24-hour forecast chart (Matplotlib)
- 🎞️ Weather animations (rain, thunder, snow)
- 🔊 Weather-based sound effects
- 💧 Humidity display
- 🎨 Dynamic UI based on weather conditions


## 🛠️ Technologies Used

- Python 🐍
- PyQt5 (GUI)
- OpenWeather API 🌍
- Matplotlib 📊
- Requests library

## ⚙️ Installation & Run

1. Clone the repository

git clone https://github.com/sharifulshihab/Weather-App-Api.git
cd Weather-App-Api

2. Install dependencies
pip install -r requirements.txt

3. Run the app
python weather_app.py


##🔑 API Key Setup

This project uses OpenWeather API.

Replace API key in code:

api_key = "YOUR_API_KEY"
